# Shimla Spice - Full Stack E-commerce

A conversion-focused React + Tailwind + Express storefront for premium homemade pickles.

## Features
- Emotional, story-first homepage with trust hooks, scarcity, social proof, and offers.
- Product pages with taste meter and sticky mobile buy CTA.
- Cart-ready state management, dashboard, and order tracking timeline.
- JWT auth APIs for signup/login.
- Checkout and tracking endpoints with dummy data.

## Tech
- Frontend: React, Vite, Tailwind, Framer Motion
- Backend: Node.js, Express, JWT
- Database layer: Mongo-ready structure with in-memory dummy data for quick start

## Run
```bash
npm install
npm install --prefix client
npm install --prefix server
npm run dev
```
- Frontend: http://localhost:5173
- Backend: http://localhost:5000
